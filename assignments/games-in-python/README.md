
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a classic Hangman game in Python. You will practice string manipulation, loops, conditionals, random word selection, and user input handling.

## 📝 Tasks

### 🛠️ Build the Core Hangman Game

#### Description
Create a playable Hangman game that selects a random word from a predefined list and lets the player guess one letter at a time.

#### Requirements
Completed program should:

- Randomly select a word from a predefined word list
- Display current word progress using underscores for unguessed letters
- Accept one-letter guesses from the user
- Reveal all matching letter positions when a correct guess is made
- Reduce remaining attempts when an incorrect guess is made

### 🛠️ Complete Game Flow and End Conditions

#### Description
Finish the game so it cleanly handles both winning and losing outcomes with clear feedback.

#### Requirements
Completed program should:

- End with a win message when the full word is guessed
- End with a lose message when attempts are exhausted
- Display the final correct word when the player loses
- Show guesses remaining after each turn
- Prevent duplicate guesses from counting as new attempts
